# E-Learning Platform UI

## Description
Front-end design for an E-Learning platform with course listings, progress tracking, and embedded videos.

## Technologies Used
- HTML5
- CSS3

## Features
- Course listing cards
- Progress tracking bars
- Video embedding
- Responsive layout

## How to Run
Open `index.html` in your browser.
